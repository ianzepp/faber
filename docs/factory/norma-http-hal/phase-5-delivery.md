# Phase 5 Delivery Spec: Docs And Expansion Pattern

## Scope

Document the reusable implementation pattern extracted from the HTTP HAL work.

## Required Changes

- Add a durable checklist for future Norma HAL modules.
- Cover each required surface:
  - Faber interface,
  - Rust runtime module,
  - runtime exports,
  - Cargo dependencies,
  - Rust codegen bridge,
  - concrete runtime type mapping,
  - runtime tests,
  - package fixture,
  - user-facing docs/explain follow-up.
- Update this factory plan's status/progress notes so the run is resumable.

## Checkpoint

The docs must identify enough steps for a future HAL module to follow the HTTP
implementation path without rereading all code changes.
